# Fonts Directory

This directory contains the default font used by Tartrazine's image formatters (PNG, JPEG, WebP).

## Tiny-Regular.ttf

- **Actual Font**: JetBrains Mono
- **Version**: Rebranded as "Tiny" from tinymanorg/brand repository
- **License**: SIL Open Font License 1.1 (OFL-1.1)
- **License File**: [OFL.txt](./OFL.txt)
- **Original Source**: https://github.com/JetBrains/JetBrainsMono
- **File Size**: 69KB

## Historical Note

The PCF fonts (*.pcf.gz files in this directory) were used by the old
PNG formatter implementation and have been replaced by this TrueType font.
